## 下载

- macOS Apple Silicon: `FlowDesk_V0_macOS-arm64.dmg`
- macOS Intel: `FlowDesk_V0_macOS-Intel.dmg`
- Windows x64: `FlowDesk_V0_windows-x64.zip`

## 主要能力

- 14 天全功能试用
- 离线激活码
- 本地 SQLite 数据库，数据保存在本机
- 自动编号
- 保存任务后自动创建任务文件夹
- 今日 / 本周 / 一本帐 / 月历 / 看板视图
- Excel / CSV / JSON 导入导出
- 专业版 AI 助手配置入口

## 安装提示

- Windows 版为便携 zip，解压后运行 `FlowDesk.exe`。
- macOS 版如遇系统安全提示，请在“系统设置 - 隐私与安全性”中允许打开。
- 文件完整性可用本 Release 附带的 `checksums.txt` 校验。

应用内自动更新会在后续补齐 Tauri updater artifacts、签名和 `latest.json` 后启用。
